# 联系人工具箱 iOS App


## 功能

- 表格转 VCF：从系统“文件”App 选择 `.csv` / `.xlsx`，清洗后生成 `contacts.vcf`
- VCF 转 Excel：从系统“文件”App 选择 `.vcf`，解析联系人后导出 `.xlsx` 或 `.csv`
- 通讯录导出：申请系统通讯录权限，选择手机联系人后导出 VCF / Excel / CSV
- 所有生成文件均通过 iOS 系统分享面板保存或发送到其他应用
- 自动识别姓名、手机号、邮箱、公司、地址、备注
- 支持字段映射、国家码、手机号去重、异常行提示
- 预览默认展示 5 条，可进入全部联系人页面查看完整信息
- 支持自定义导出文件名，自动按格式补齐扩展名
- 支持中文/英文双语，中文系统显示中文，其他系统语言默认显示英文
- 已配置 AppIcon 资源和 LaunchScreen 启动页
- 内置联系人导入模板页面


也可以直接用 Xcode 打开 `ContactTool.xcodeproj` 运行。
